# Laravel Livewire 3 Skill

An AI agent skill for building reactive server-driven UI with Livewire 3, Alpine.js, and Volt.

## Topics Covered

- Component classes with URL-synced properties and computed properties
- Forms with real-time validation and wire:model modifiers
- Actions, events, and browser dispatch
- File uploads with temporary preview URLs
- Lazy loading and performance optimization
- Volt single-file components
- Wire:loading states and wire:confirm

## Installation

```bash
npx skills add neuraliq/laravel-livewire
# or
php artisan boost:add-skill neuraliq/laravel-livewire
```

## Compatibility

Claude Code, Cursor, Windsurf, GitHub Copilot

## Requirements

Laravel 11+, Livewire 3, Alpine.js
